# SQL Sales Data Analysis

Tools Used:
- SQL
- SQLite / MySQL / PostgreSQL

Project Goal:
Analyze sales data using SQL queries to extract meaningful business insights.

Dataset:
- sales_data.csv (100 rows of sample sales transactions)

Key Questions Answered:

1. What is the total revenue?
2. Which region generates the most revenue?
3. Which product category performs best?
4. Who are the top customers?
5. What are the top selling products?
6. How does revenue change month by month?
7. What is the average order value?

SQL Skills Demonstrated:
- Aggregations (SUM, AVG, COUNT)
- GROUP BY
- ORDER BY
- Filtering
- Date-based analysis

Files in Repository:

data/sales_data.csv  
sql/analysis_queries.sql  

How to Use:

1. Load the CSV into any SQL database
2. Run the queries from analysis_queries.sql
3. Review the results to understand business performance
